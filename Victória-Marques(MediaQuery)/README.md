-> 👩‍🚀O projeto foi feito com a intenção de chamar a atenção dos clientes para que eles possam explorar mundos novos, conseguindo pagar suas passagens sem nenhum tipo de problema ou seja podendo ir ao ☄️INFINITO e ALÉM!!! tendo as passagens para as estrelas. Sendo feito atráves do Front-End.

#Explicação do Problema:

-> 🌌O problema no layout que estava sendo quebrado o tempo todo, foi resolvido atráves do código chamado Media Query do Tablet e do Smartphones assim colocando o tamanho certo e chamando as classes de cada um desses elementos do design responsivo, pois voltou à funcionar normalmente como qualquer outro site.
